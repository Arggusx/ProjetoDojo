## Guia
### Rode esses comandos no terminal
```
git clone https://github.com/EricGusmao/ProjetoDojo.git
cd ProjetoDojo
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```
### Execute o projeto
```
# Na pasta do projeto
python run.py
```
